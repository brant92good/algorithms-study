How to study this這該怎麼念 (~1 week out)（大概剩一週）

Don't read-all-then-test — that's the fluency-illusion trap. Per chapter: learn → immediately self-test → mix older chapters into later sessions (retrieval + spacing + interleaving). NP-completeness is pure reproduce-the-reduction practice, so drill it actively.別等全部讀完才測驗 — 那樣很容易掉進 fluency-illusion（誤以為自己會了）的陷阱。每一章都這樣做：先學 → 馬上自己測一遍 → 後面複習時再把前面的章節穿插進來（也就是 retrieval + spacing + interleaving）。NP-completeness 說穿了就是一直練怎麼重現 reduction，所以更要動手做。
