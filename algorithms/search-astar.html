A* Search

Best-first search that finds the cheapest path to a goal — by always expanding the node whose guessed total cost f = g + h is smallest, with a guess that never overestimates.

What is it?

You are searching a tree of partial solutions, starting at the root and trying to reach a goal node as cheaply as possible. Plain best-first search expands whichever open node looks best and stops at the first goal it reaches — but that goal need not be the cheapest one. A* fixes this with a careful cost estimate.

For every node n, A* tracks three numbers:

  • g(n) — the actual cost already paid to get from the root down to n (the sum of edge weights on the path so far). This is known exactly.
  • h(n) — a heuristic estimate of the cost still remaining from n to a goal. This is a guess.
  • h*(n) — the true (best possible) remaining cost from n to a goal. We usually don't know it, but it's what h is trying to approximate.

A* combines the known and the guessed into f(n) = g(n) + h(n): an estimate of the total cost of the best solution that passes through n. At each step it removes the open node with the smallest f, expands it (generates its children, computing their g/h/f), and repeats.

Admissibility. A* is guaranteed to find the optimal goal as long as the heuristic never overestimates: h(n) ≤ h*(n) for every node. When that holds, f(n) = g(n) + h(n) ≤ g(n) + h*(n) = f*(n), so f(n) is a lower bound on the true best total cost through n. That lower bound is the key to optimality.

Why optimality follows. A* stops only when the node it selects for expansion (the smallest-f open node) is itself a goal. Say that goal is reached with cost f = g (a goal has h = 0). Every other open node n has f(n) ≥ that value, and since f(n) is a lower bound, no path through any other open node could finish more cheaply. So the first goal A* selects is optimal. (Just seeing a goal in the open list isn't enough — it has to be the one picked next.)

Tiny example. If a node has g = 4 (already spent 4) and h = 1 (guessing 1 more), then f = 5. A sibling with g = 3, h = 2 also has f = 5 — A* treats them as equally promising and breaks the tie arbitrarily.
