Satisfiability (SAT)

Given a Boolean formula, is there any true/false assignment to its variables that makes the whole thing true? Search the tree of 2ⁿ assignments — and prune the instant a clause is broken.

📄 Source: Alg05 · The Searching Strategies §1 (the SAT problem; tree of 2ⁿ assignments; partial tree with early pruning).

🧠 Solved by: Depth-First Search (and BFS)

The problem

A Boolean variable is one that is either true or false. A literal is a variable or its negation — e.g. x₁ or ¬x₁ (read "not x₁"). A clause is an OR of literals, e.g. (x₂ ∨ x₅); it is satisfied if at least one of its literals is true. A formula is in conjunctive normal form (CNF) when it is an AND of clauses — so it is true only when every clause is satisfied at once.

The satisfiability problem asks: given such a CNF formula in n variables x₁ … xₙ, does there exist a truth assignment that makes the formula true? If yes, the formula is satisfiable; if no assignment works, it is unsatisfiable.

The tiny instance (from the slide). Five clauses over variables x₁ … x₅, all of which must hold simultaneously:

(1) ¬x₁ ← x₁ must be false (2) x₁ ← x₁ must be true (3) x₂ ∨ x₅ (4) x₃ (5) ¬x₂

Look at clauses (1) and (2): one demands x₁ = false, the other demands x₁ = true. They flatly contradict, so no assignment can satisfy both — this instance is unsatisfiable. The whole point of §1 is that we can discover this without enumerating all assignments: the very first variable we branch on already breaks a clause on both sides.

With n variables there are 2ⁿ possible assignments. We lay them out as a binary tree of depth n: at level i we branch on xᵢ = T (left) versus xᵢ = F (right). Every root-to-leaf path is one complete assignment; the leaves are the 2ⁿ rows of the truth table.

Watch it solved

We run depth-first search down the 2ⁿ-assignment tree, checking after each choice whether any clause is already falsified by the partial assignment. The moment one is, that whole subtree is dead — prune it and back up. Step through.

Primary source: Alg05 · The Searching Strategies §1 — the SAT problem, the "Tree Representation of Eight Assignments" (the full 2³ truth-table tree), and "A Partial Tree to Determine the Satisfiability Problem," whose message is: we may not need to examine all 2ⁿ assignments — a partial assignment that violates a clause kills its whole subtree. SAT is the historical seed of NP-completeness (Cook–Levin).

The method: this tree is explored by Depth-First Search (stack / dive-and-backtrack) or Breadth-First Search (queue / level-by-level) — same tree, different visiting order. Sibling problems in this chapter: Hamiltonian Circuit and Sum of Subsets — both also cast as searching a solution-space tree with early pruning of dead branches.
