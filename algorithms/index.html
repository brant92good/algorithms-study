🔎 Searching Strategies — Chapter 6: explore a solution tree

✓
Depth-First Search
Dive deep with a stack; backtrack on a dead end.
 ✓
Breadth-First Search
Same tree, a queue, level by level.
 ✓
Hill Climbing
Greedy local choice via f(n)=d(n)+w(n) — can get stuck.
 ✓
Best-First Search
Pick the best node among ALL open ones — global view.
 ✓
Branch-and-Bound
Add a bound that prunes losing branches → optimization (TSP).
 ✓
A* Algorithm
f=g+h with admissible h≤h* → first goal popped is optimal.
