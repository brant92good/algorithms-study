⛰️ Heuristic Search — follow an evaluation function f(n)

Methods: Hill Climbing (greedy, local — can stall) · Best-First Search (global open list — recovers)

✓
The 8-Puzzle
Slide tiles to the goal, scored by f = d + w — solved both ways, hill-climbing vs best-first.

✂️ Branch-and-Bound — best-first + a bound that prunes losing branches → optimization

Method: Branch-and-Bound

✓
Personnel Assignment
Assign people to precedence-ordered jobs at minimum total cost.
 ✓
Travelling Salesperson
Cheapest tour visiting every city once (reduced-matrix bound → cost 126).
 ✓
0/1 Knapsack
Max-profit packing within capacity — bounded & pruned.

⭐ A* Search — f(n) = g(n) + h(n) with an admissible heuristic

Method: A* Algorithm

✓
Shortest Path with A*
Least-cost path to a goal; admissible h ≤ h* ⇒ first goal popped is optimal.
 ✓
Channel Routing
Wire all nets across a channel using the fewest tracks (NP-complete).

🪓 Prune-and-Search — discard a constant fraction each round → O(n)

Method: Prune-and-Search

✓
Selection (k-th Smallest)
The k-th smallest element in linear time (median of medians).
 ✓
2-Variable Linear Programming
Minimize a linear objective under linear constraints — the F(x) envelope.
 ✓
1-Center
The smallest circle enclosing all the points.

🧩 Dynamic Programming — solve overlapping subproblems once, reuse them

Method: Dynamic Programming

✓
Multistage Graph
Min-cost path through a staged graph (forward & backward).
 ✓
0/1 Knapsack

 ✓
Resource Allocation
Split units across projects to maximize total profit.
 ✓
Travelling Salesperson (DP)
Cheapest tour via Held–Karp g(i,S); O(n²·2ⁿ).
 ✓
Longest Common Subsequence
Longest subsequence shared by two strings.
 ✓
Optimal Binary Search Tree
The BST with the lowest expected search cost.
