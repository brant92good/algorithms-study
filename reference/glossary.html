Ch6 · Searching StrategiesCh6 · Searching 策略

Solution tree
Tree whose root-to-leaf paths enumerate all candidate solutions; each level fixes one decision. Searching = walking this tree.一棵樹，其 root-to-leaf 路徑列舉出所有候選解；每一層固定一個決策。Searching 就是走訪這棵樹。
DFS (depth-first search)
Expand the deepest unexpanded node first. Implemented with a stack. Backtracks on dead ends.先展開最深的未展開節點。用 stack 實作。遇到死路就 backtrack。
BFS (breadth-first search)
Expand the shallowest unexpanded node first. Implemented with a queue.先展開最淺的未展開節點。用 queue 實作。
Hill climbing
DFS variant: among the children of the current node, expand the locally best one (greedy). No global view — can stall in a local optimum.DFS 的變體：在目前節點的子節點中，展開局部最佳的那一個（greedy）。沒有全域視野——可能卡在 local optimum。
Best-first search
Expand the node with the best estimate among all open nodes (global view). Combines DFS + BFS reach.在所有 open 節點中展開估計值最佳的那一個（全域視野）。結合了 DFS 與 BFS 的觸及範圍。
Evaluation function f(n)
Estimated promise of node n. 8-puzzle: f(n)=d(n)+w(n), d=depth, w=# misplaced tiles.節點 n 的估計潛力。8-puzzle：f(n)=d(n)+w(n)，d=深度，w=放錯位置的方塊數。
Branch-and-bound
Strategy for optimization problems: prune any subtree whose bound can't beat the best solution found. (DFS/BFS/hill/best-first solve only feasibility, not optimization.)用於optimization問題的策略：修剪任何 bound 無法勝過目前最佳解的子樹。（DFS/BFS/hill/best-first 只解 feasibility，不解 optimization。）
Lower bound (reduced cost matrix)
Subtract the row min from each row, then the column min from each column, so every row & column has a 0. The total subtracted is a lower bound on any tour cost.每一列減去該列的最小值，再讓每一行減去該行的最小值，使每一列與每一行都有一個 0。減去的總和就是任何 tour 成本的 lower bound。
A* algorithm
Best-first for optimization with f(n)=g(n)+h(n): g=cost root→n, h=estimate n→goal. If h(n)≤h*(n) (never overestimates), the first goal popped is optimal.用於 optimization 的 best-first，f(n)=g(n)+h(n)：g=root→n 的成本，h=n→goal 的估計值。若 h(n)≤h*(n)（永不高估，即 admissible），第一個被取出的 goal 即為最佳解。

Prune-and-SearchPrune-and-Search

Prune-and-search
Each iteration discards a fixed fraction f of the data, then recurses on the rest. T(n)=T((1−f)n)+O(n^k) ⇒ O(n^k).每次疊代丟棄資料中固定比例 f，再對其餘部分遞迴。T(n)=T((1−f)n)+O(n^k) ⇒ O(n^k)。
Selection problem
Find the k-th smallest of n elements. Median = the (n/2)-th smallest.在 n 個元素中找出第 k 小的元素。Median（中位數）= 第 (n/2) 小的元素。
Median of medians
Split into groups of 5, take each group's median, take the median of those as pivot p. Guarantees ≥ n/4 pruned per step ⇒ selection in O(n).分成每組 5 個，取每組的 median，再取這些 median 的 median 當作 pivot p。保證每一步至少修剪 ≥ n/4 ⇒ selection 為 O(n)。

Ch7 · Dynamic ProgrammingCh7 · Dynamic Programming

Multistage graph
DAG with vertices partitioned into ordered stages V₁…V_k; edges go stage i→i+1. Goal: min-cost path s→t.一個 DAG，其頂點被切分成有序的 stage V₁…V_k；邊只從 stage i→i+1。目標：s→t 的最小成本路徑。
Principle of optimality
If a sequence of decisions D₁…Dₙ is optimal, then every suffix (last k decisions) is itself optimal. Lets us write a recurrence.若一連串決策 D₁…Dₙ 是最佳的，則每個後綴（最後 k 個決策）本身也是最佳的。這讓我們能寫出 recurrence。
Forward reasoning
Express decision xᵢ via optimal sequences for x_{i+1}…xₙ. (Relations formed forward are solved backward.)用 x_{i+1}…xₙ 的最佳序列來表示決策 xᵢ。（往前建立的關係式，往後求解。）
Backward reasoning
Express decision xᵢ via optimal sequences for x₁…x_{i−1}.用 x₁…x_{i−1} 的最佳序列來表示決策 xᵢ。
0/1 knapsack (DP)
gᵢ(y)=max{ g_{i+1}(y), g_{i+1}(y−w_{i+1})+p_{i+1} }; answer g₀(M).gᵢ(y)=max{ g_{i+1}(y), g_{i+1}(y−w_{i+1})+p_{i+1} }；答案為 g₀(M)。

Ch8 · NP-CompletenessCh8 · NP-Completeness

P
Problems solvable by a deterministic polynomial-time algorithm.能由deterministic（確定性）多項式時間演算法求解的問題。
NP
Decision problems solvable by a nondeterministic polynomial algorithm = guess a certificate, check it in poly time.能由nondeterministic（非確定性）多項式演算法求解的decision問題 = 猜一個 certificate，再在多項式時間內檢查它。
Reduction A → B
A solvable in poly time using a poly-time algorithm for B. "A is no harder than B."利用 B 的多項式時間演算法，就能在多項式時間內解 A。「A 不會比 B 難。」
NP-hard
Every NP problem reduces to it. (May not be in NP — e.g. the halting problem.)每個 NP 問題都能 reduce 到它。（它本身不一定在 NP 內——例如 halting problem。）
NP-complete (NPC)
NP-hard and in NP. If any NPC problem ∈ P then P = NP.既是 NP-hard 又在 NP 內。只要有任何一個 NPC 問題 ∈ P，就會推得 P = NP。
Nondeterministic algorithm
Two phases: choice() guesses; then a deterministic check. NP if the check is polynomial.兩個階段：choice() 進行猜測；接著做 deterministic 檢查。若檢查是多項式時間，即屬於 NP。
Cook's theorem
SAT is NP-complete — the first one. Every NP problem reduces to SAT.SAT 是 NP-complete——而且是第一個被證明的。每個 NP 問題都能 reduce 到 SAT。
Decision vs optimization
Decision answers Yes/No ("is there a tour ≤ C?"). Solve optimization by binary-searching C with the decision algorithm.Decision 回答 Yes/No（「是否存在 ≤ C 的 tour？」）。對 C 做 binary search 並搭配 decision 演算法，即可解 optimization。
